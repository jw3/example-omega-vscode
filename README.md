example omega edit extension
===

Example VS Code plugin that uses embedded omega bindings to generate content.

### requires
- bindings compiled against Electron Node v14.16.0
- VS Code v1.60-1.63

### run

```
yarn
yarn package
code --extensionDevelopmentPath=<this-project-dir>
```
